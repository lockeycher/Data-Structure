Setup and submission

Place written answers into a text, Word, or PDF document with the name finalexam followed by the appropriate extension. Place all Java code into an Eclipse package called finalexam. Zip the package into a file called finalexam.zip and submit that single file. Everything means:

  1. the package folder finalexam
  2. the document finalexam.{txt,doc,docx,pdf} with the written answers
  3. HashTable.java
  4. DFSTrace.java
  5. BFSTrace.java
I have the test programs so please do not include those.

Programming problems

6. (10 points) The class HashTable is an implementation of a hash table. Add a method called keys that returns an Iterable object with the keys in the hash table sorted in ascending order. You may test it with the program TestHashTable. Both are found in HashTableCode.zip.

7. (10 points) Write a program DFSTrace that contains a version of a depth first search that prints a trace of its traversal. Write a public static method: 
public static void dfsPrintTrace(Graph g) {
    // *** Declare and initialize the marked array.
    dfsPrintTrace(g, 0, marked, 0);
}
This method calls a private method: 
private static void dfsPrintTrace(Graph g, int start, boolean[] marked, int indent)
All references to a method below refer to this second method. Every message printed should be preceded by the number of spaces specified by the indent parameter. A recursive call should pass an indent value with 1 added.
  • The first time the method visits a vertex, it should print the message "First time at vertex n", where n is the integer label of the vertex.
  • Every other time it visits a vertex, it should print the message "Visiting vertex n again".
  • When it exits, it should print the message "Finished searching from n".

Here's the output when run on the file data/tinyG.txt:

  First visit to 4.

     Visited 4 again.
    Finished with 3.
    Visited 4 again.
    Visited 0 again.
   Finished with 5.
   Visited 6 again.
   Visited 3 again.
  Finished with 4.
 Finished with 6.
 First visit to 2.
  Visited 0 again.
 Finished with 2.
 First visit to 1.

 Finished with 1.

Here's the main method to insert into your program to test your method(s): 

	public static void main(String[] args) {
		In input = new In("data/tinyG.txt");
		Graph g = new Graph(input);

		dfsPrintTrace(g);
	}

Include in the comments of your code the answer to this question: What is the pattern formed by the "First visit" and "Finished" messages?
